‘Word Race’ is a game, designed to improve QWERTY typing rate and efficiency.

*This website is created using simple HTML, CSS and Js.

**Instructions:

*The game will automatically begin once you start typing the provided paragraph.
*The timer will also start once you started typing.
*Game will be auto completed when the time ends.
*You can click on 'Try Again' to open a next para.
